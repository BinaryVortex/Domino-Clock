# 🁢 Domino Clock

A creative digital clock built with a **domino-inspired display**, where each digit is represented using domino tiles instead of traditional seven-segment displays. The project combines clean UI design with smooth time updates to create a unique way of visualizing the current time.

> A fun front-end project showcasing JavaScript logic, DOM manipulation, and creative UI design.

---

## ✨ Features

- 🕒 Real-time clock that updates automatically
- 🎲 Domino-style digit representation
- ⚡ Smooth and lightweight
- 📱 Responsive layout
- 🎨 Clean, modern interface
- 🌐 Runs entirely in the browser

---

## 🚀 Getting Started

### Prerequisites

- Modern web browser
- VS Code (recommended)

### Installation

```bash
git clone https://github.com/BinaryVortex/Domino-Clock.git
cd Domino-Clock
```

Open `index.html` directly in your browser, or launch a local server.

### Using VS Code Live Server

1. Install **Live Server**.
2. Open the project.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## ⚙️ How It Works

Every second, JavaScript retrieves the current system time, converts each numeral into its domino equivalent, and updates the UI dynamically. The mapping logic separates presentation from time calculation, making the project easy to extend and customize.
